Free mode: By default this app uses browser-only parsing, the free MiniAI v6 local classifier, name/gender/country guessing, optional OCR for scanned PDFs/images, Word/PDF/Excel pattern rules, and a confidence filter that removes rows below 60%. No API key or paid quota is needed.

MiniAI v6 model: This is not a cloud LLM. It is a larger local Naive-Bayes-style scoring model made from extended name dictionaries, surname hints, suffix rules, script detection, and country patterns.

Privacy note: If paid AI mode is enabled, passenger text is sent to the configured API endpoint. For production use with real TR ID/passport data, use a private backend proxy instead of exposing an API key in the browser.
